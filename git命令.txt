项目第一次提交
1.初始化本地仓库
git init
2.全部提交本地仓库
git add .

(查看自己提交了什么)
git status
3.指向到远程仓库 (第一次必须的) 
git remote add origin https://github.com/SNCTTOO/newGi.git
4.提交到远程仓库
git push origin master

.gitignore 作用 忽略不要提交


----------------------------------------

第二次提交 跟第一次提交一样 少了指向远程仓库这一步
git pull origin master //从Github上pull到本地源码库
分支
一个项目多人开发 开发不同模块的
branch 分支

查看本地分支 (master--主分支)

在本地创建分支
git branch develop

切换分支
git checkout 分支名

----------------------------------------

合并分支


01.回到本地主分支
git checkout master

02.合并分支
git merge 分支名
